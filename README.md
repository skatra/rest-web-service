Feel free to connect:<br>
[stevekatra.com](https://www.stevekatra.com/) <br>

## Setup for Intellij
```
gradle cleanIdea idea
```

## Build
```
gradle clean build
```

## Run
```
gradle bootRun
```

## Swagger2 Documentation
```
http://localhost:8080/swagger-ui.html
```
